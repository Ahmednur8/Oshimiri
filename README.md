
# Oshimiri 🌊

> *Oshimiri — from Igbo tradition, meaning "The River That Never Runs Dry." A source of strength that never fails.*

AI coding assistant powered by Oshi. Generate and debug production-ready code from simple prompts. Humanity first approach to AI.

## Try It Now

🌐 **Cloud (instant, no setup):** [oshimiri.netlify.app](https://oshimiri.netlify.app)

## What Oshi Can Build

From short prompts Oshi generates production-ready code across 11 languages:

- REST APIs and FastAPI backends
- Authentication systems with JWT
- Neural networks and ML models
- Blockchain with proof of work
- Drone control systems
- End-to-end encrypted messaging
- Mobile applications
- Desktop IDEs
- Smart contracts
- Multi-agent systems

## Two Modes

**☁️ Cloud** — Fast, accessible, powered by Llama 3.3 70B via Groq. Anyone can use it instantly.

**🖥️ Local** — Completely private. Your code never leaves your machine. Runs on Ollama.

## Run Locally
```bash
# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Pull a model
ollama pull qwen2.5-coder:7b

# Install dependencies
pip install fastapi uvicorn ollama

# Download and run the local backend
python3 oshimiri_local.py
```

Then open [oshimiri.netlify.app](https://oshimiri.netlify.app) and click 🖥️ Local.
